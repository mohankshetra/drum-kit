🥁 Drum Kit Web App

An interactive Drum Kit web application that allows users to play drum sounds using both mouse clicks and keyboard keys.

---

🚀 Features

- 🎹 Play drum sounds using keyboard keys ("W A S D J K L")
- 🖱️ Click on drum images to play sounds
- ✨ Visual feedback: button glows on press
- 🔊 Realistic drum sound effects
- 📱 Responsive and user-friendly UI

---

🎮 How to Use

1. Open the app in your browser
2. Press any of the following keys:

W A S D J K L

OR

👉 Click on the drum images

✔️ You will hear a drum sound
✔️ The button will animate (glow effect)
